# Jadwala University 

An intelligent University Timetable Management System aimed at solving scheduling conflicts between lecture halls and professor availability through automated timetable generation.

## ✨ Key Features
- Automated Scheduling: Utilizes smart algorithms to generate timetables while preventing room and time conflicts.
- Professor & Room Management: Track professor availability and classroom capacity in real-time.
- Multi-Portal Access: Dedicated dashboards for Admins to manage data and for Students to view their schedules.
- Notification System: Keeps users updated on any scheduling changes.
- Export Options: Download timetables as print-ready PDF files.
- Backup & Restore: Integrated system for data persistence and version control of previous schedules.

## 🚀 Tech Stack
- Frontend: React, TypeScript, Vite.
- Styling: Tailwind CSS, shadcn/ui.
- Backend: Supabase (Auth, Database, Edge Functions for Schedule Generation).
- State Management: TanStack Query (React Query).

## 🛠 Local Setup
1. git clone https://github.com/amar-eisa/jadwala-uni.git
2. npm install
3. npm run dev
