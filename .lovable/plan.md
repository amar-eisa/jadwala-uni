

## تحويل القائمة الجانبية إلى شريط مهام سفلي (Taskbar)

### الفكرة
استبدال القائمة الجانبية بشريط سفلي ثابت يشبه شريط مهام Windows، مع الحفاظ على جميع الوظائف الحالية. هذا يعطي التصميم طابعاً مخصصاً ومميزاً.

### التصميم

```text
┌─────────────────────────────────────────────────────┐
│  Header: شعار + ThemeToggle + NotificationBell      │
├─────────────────────────────────────────────────────┤
│                                                     │
│                  المحتوى الرئيسي                     │
│                                                     │
│                                                     │
├─────────────────────────────────────────────────────┤
│  ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐ ┌──┐  ···  👤 ⚙️  │
│  │📊│ │🚪│ │🎓│ │👥│ │📚│ │🕐│ │📅│       │خروج│  │
│  └──┘ └──┘ └──┘ └──┘ └──┘ └──┘ └──┘       └───┘  │
└─────────────────────────────────────────────────────┘
```

### التغييرات — ملف واحد: `src/components/Layout.tsx`

1. **إزالة القائمة الجانبية** بالكامل (`<aside>`)
2. **إضافة شريط سفلي ثابت** (`fixed bottom-0`) بتصميم glassmorphism:
   - ارتفاع ~64px مع `backdrop-blur` و `rounded-t-2xl`
   - أيقونات التنقل الأساسية في المنتصف (7 عناصر رئيسية)
   - قائمة "المزيد" (⋯) تفتح popup لباقي العناصر (التقارير، سجل النشاطات، الإعدادات، الإدارة)
   - أفاتار المستخدم + زر الخروج في الطرف الأيسر
   - الحفاظ على حركة `nav-pill` (motion.div layoutId) للعنصر النشط
3. **تعديل Header**: يصبح دائماً ظاهراً (وليس mobile فقط) — يحتوي الشعار + شعار الجامعة + ThemeToggle + NotificationBell
4. **إزالة `lg:pr-80`** من المحتوى الرئيسي وإضافة `pb-20` لمنع تداخل المحتوى مع الشريط السفلي
5. **الموبايل**: نفس الشريط السفلي مع أيقونات أصغر وبدون نصوص، والقائمة الإضافية تفتح لأعلى

### التفاصيل التقنية
- العناصر الرئيسية (7): لوحة التحكم، القاعات، الدكاترة، المجموعات، المواد، الفترات، الجدول
- العناصر الثانوية (في قائمة "المزيد"): التقارير، سجل النشاطات، الإعدادات، الإدارة/المراقبة
- tooltip يظهر اسم العنصر عند التمرير على الأيقونة (desktop)
- العداد (count badge) يظهر كنقطة صغيرة فوق الأيقونة بدلاً من رقم كامل
- الشريط يستخدم نفس ألوان sidebar الحالية للحفاظ على التناسق
