
# إصلاح عرض القائمة الجانبية والتمرير

## المشكلة
القائمة الجانبية لا تظهر كاملة ولا يمكن التمرير لعرض جميع العناصر. السبب هو:
1. منطقة التنقل لا تحتوي على خاصية التمرير
2. قسم معلومات المستخدم في الأسفل (position: absolute) يغطي جزءاً من عناصر التنقل

---

## الحل

### تعديل هيكل القائمة الجانبية (Layout.tsx)

**التغييرات المطلوبة:**

| المكون | التغيير |
|--------|---------|
| `aside` | إضافة `flex flex-col` لتنظيم العناصر عمودياً |
| `nav` | إضافة `flex-1 overflow-y-auto` للتمرير وملء المساحة المتاحة |
| قسم المستخدم | تغيير من `absolute` إلى `sticky` أو إزالة الـ absolute وجعله جزء من التدفق الطبيعي |

**البنية الجديدة:**

```text
┌──────────────────────────┐
│  شعار جدولة + الجامعة   │  ← ثابت في الأعلى
├──────────────────────────┤
│                          │
│  عناصر التنقل           │  ← قابل للتمرير (flex-1 overflow-y-auto)
│  - لوحة التحكم          │
│  - القاعات              │
│  - الدكاترة             │
│  - المجموعات            │
│  - المواد               │
│  - الفترات الزمنية      │
│  - الجدول               │
│  - الإعدادات            │
│  - الإدارة              │
│                          │
├──────────────────────────┤
│  معلومات المستخدم       │  ← ثابت في الأسفل (mt-auto)
│  + زر تسجيل الخروج      │
└──────────────────────────┘
```

---

## التغييرات في الكود

### 1. إضافة Flexbox للـ aside
```tsx
<aside className="... flex flex-col">
```

### 2. تعديل منطقة التنقل
```tsx
<nav className="flex-1 flex flex-col gap-1 p-4 overflow-y-auto">
```

### 3. تعديل قسم المستخدم
```tsx
// من:
<div className="absolute bottom-0 left-0 right-0 p-4 ...">

// إلى:
<div className="mt-auto p-4 border-t border-sidebar-border shrink-0">
```

## النتيجة المتوقعة

- جميع عناصر القائمة ستظهر بشكل كامل
- يمكن التمرير لأسفل لرؤية جميع العناصر على الشاشات الصغيرة
- قسم معلومات المستخدم يبقى ثابتاً في الأسفل دون أن يغطي عناصر التنقل
