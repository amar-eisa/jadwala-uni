

# 🎓 نظام جدولة الجامعة (University Timetabling System)

## نظرة عامة
نظام ذكي لإنشاء جداول المحاضرات الجامعية تلقائياً مع تجنب التعارضات وتوزيع الحمل على القاعات بشكل متوازن.

---

## 📊 الصفحات والميزات الرئيسية

### 1. لوحة التحكم الرئيسية (Dashboard)
- نظرة سريعة على إحصائيات النظام (عدد القاعات، الدكاترة، المجموعات، المواد)
- أزرار سريعة للوصول لأقسام الإدارة المختلفة
- حالة آخر جدول تم توليده

### 2. إدارة القاعات (Rooms)
- إضافة/تعديل/حذف القاعات
- تحديد نوع القاعة (معمل Lab / قاعة محاضرات Lecture)
- قائمة بكل القاعات المتاحة

### 3. إدارة الدكاترة (Professors)
- إضافة/تعديل/حذف الدكاترة
- عرض قائمة الدكاترة مع موادهم

### 4. إدارة مجموعات الطلاب (Student Groups)
- إضافة/تعديل/حذف مجموعات الطلاب
- عرض قائمة المجموعات

### 5. إدارة المواد الدراسية (Subjects)
- إضافة مواد جديدة مع ربطها بالدكتور والمجموعة
- توليد كود المادة تلقائياً من النظام
- عرض قائمة المواد مع تفاصيلها

### 6. إدارة الفترات الزمنية (Time Slots)
- إضافة/تعديل/حذف الفترات الزمنية
- تحديد اليوم ووقت البداية والنهاية
- مرونة كاملة في تخصيص الفترات حسب احتياجات الجامعة

### 7. توليد الجدول (Generate Schedule)
- زر لتوليد الجدول تلقائياً
- خوارزمية ذكية تتجنب:
  - ✅ تعارض القاعات (نفس القاعة في نفس الوقت)
  - ✅ تعارض الدكاترة (نفس الدكتور في نفس الوقت)
  - ✅ تعارض المجموعات (نفس المجموعة في نفس الوقت)
- توزيع الحمل بالتساوي على القاعات (Least Used First)

### 8. عرض الجدول (Timetable View)
- عرض أسبوعي (Week View) بتصميم Grid
- الأيام كأعمدة والفترات الزمنية كصفوف
- ألوان مميزة لكل مادة/دكتور
- إمكانية تصفية الجدول حسب القاعة/الدكتور/المجموعة

---

## 🏗️ البنية التقنية

### قاعدة البيانات (Supabase PostgreSQL)
- جداول: rooms, professors, student_groups, subjects, time_slots, schedule_entries
- العلاقات بين الجداول محددة بشكل صحيح
- توليد كود المادة تلقائياً باستخدام Database Trigger

### خوارزمية الجدولة (Edge Function)
- دالة TypeScript تعمل على Supabase Edge Functions
- تطبق منطق Round-Robin/Least Used First
- تتحقق من كل التعارضات قبل الإدخال

### الواجهة الأمامية (React)
- تصميم نظيف وبسيط باستخدام shadcn/ui
- جدول أسبوعي مخصص بـ CSS Grid
- نماذج إدخال سهلة الاستخدام
